# Landing Reconexion

Base estatica para construir una landing page. La estructura queda lista para
recibir el diseno y el contenido final.

## Estructura

- `index.html`: secciones principales de la landing.
- `src/styles.css`: estilos responsive y variables visuales.
- `src/main.js`: interacciones basicas del formulario y ano del footer.
- `assets/`: imagenes, iconos y recursos visuales.

## Como verla

Abre `index.html` directamente en el navegador. No requiere instalacion ni
servidor local.

## Pendiente cuando llegue el diseno

- Reemplazar paleta, tipografias, espaciados e imagen principal.
- Ajustar orden y cantidad de secciones.
- Sustituir textos placeholder por contenido final.
- Conectar el formulario a WhatsApp, email, CRM, calendario o backend.
